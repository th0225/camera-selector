# Pixel Scope

使用SvelteKit計算相機搭配不同鏡頭的視野和精度

---

## ✨ 功能特色

- 視野範圍自動計算（mm / inch）
- 像素精度自動計算（µm / inch）
- 支援遠心鏡頭和一般鏡頭
- 小數自動格式化（預設至小數點後 6 位）
- 即時反應的使用者介面（SvelteKit + TypeScript）

---
